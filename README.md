# Python-Terminal-Game-CH-
## Cthulhu Hunt

*Cthulhu Hunt* is a text-based terminal adventure game inspired by classic procedural games such as *Hunt the Wumpus* and *Zork*. It builds on these foundational concepts while introducing additional features and expanded gameplay, while still staying true to their original design philosophy.

### How to Run

To play the game, navigate to the project folder and run the following command in your terminal:

```bash
python game.py
